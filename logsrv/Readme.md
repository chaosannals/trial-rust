# LogServer LogClient

```bash
# udp 客户端
cargo run --bin logclient

# udp 服务端
cargo run --bin logserver

# tcp 客户端
cargo run --bin tcpclient

# tcp 服务端
cargo run --bin tcpserver
```

```bash
# 生成
cargo build --bin logclient
cargo build --bin logserver
cargo build --bin tcpclient
cargo build --bin tcpserver
```
